# README
In folder Arduino libs are folders containing source codes for Arduino. Those folders should be copied into "C:\Users\%USERNAME%\Documents\Arduino\libraries" folder.

In folder "Web app" there are files containing files for web application. For correct functionality you have to get simple-menu module for the Django.

Output folder is for storage of the outputs that were captured during the project development.

Snippets folder contains code snippets of the code that can be used. Some of them can be used for better understading of code.
